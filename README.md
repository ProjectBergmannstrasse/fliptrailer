# FlipTrailer

Trailing stop-and-reverse EA for XAUUSD M1 (MetaTrader 5). No indicators — one position always open, one flip line trailing behind price.

**Live site:** https://projectbergmannstrasse.github.io/fliptrailer/

## Files
- `index.html` — the full site: strategy, backtests, spread calculator, install guide
- `FlipTrailer_EA_Package.zip` — EA source, two presets, strategy manual (.docx), install guide

## Honest numbers
- Jan–Jul 2026 (zero-spread test): PF 5.73, $46k on $1k at 0.01 lots — trending-regime best case
- 2024 full year (real ticks): PF 1.36, 30% drawdown — the reality check
- Breakeven spread: ~70 pts. Read the spread section before doing anything.

Not financial advice. Past performance guarantees nothing.
